# Images Folder

## Hero Image

Place your hero image here with the filename: **hero-image.jpg**

### Recommended Specifications:
- **Format**: JPG, PNG, or WebP
- **Size**: 1000x800px or similar aspect ratio
- **File size**: Optimized (under 500KB recommended)
- **Filename**: `hero-image.jpg` (or update the path in Hero.jsx if using a different name)

### How to add your image:

1. Save your image file as `hero-image.jpg` (or `hero-image.png`)
2. Place it in this folder: `public/assets/images/`
3. The image will automatically appear on the home page

### Alternative formats:
If your image has a different name or format, you can:
- Rename it to `hero-image.jpg`, OR
- Update the image path in `src/components/Hero/Hero.jsx` (line 24)
